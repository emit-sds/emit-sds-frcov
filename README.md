# EMIT Fractional Cover Product (EMITL2BFRCOV) Code Repository and Documentation

This repository contains the following documentation related to the EMIT Level 2B Fractional Cover (L2BFRCOV) product:

- [Algorithm Theoretical Basis Document (ATBD)](docs/atbd.md)
- [User Guide](docs/user_guide.md)

In addition, the repository contains code used to generate the L2BFRCOV product, which is available via the LP DAAC (https://lpdaac.usgs.gov/).
